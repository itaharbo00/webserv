What is Webserv?

Webserv is a lightweight HTTP/1.1 web server implementation in C++, created as a 42 school project.

Technical Details

  • Language: C++98
  • Protocol: HTTP/1.0 and HTTP/1.1
  • Architecture: Non-blocking I/O with poll()
  • Features: Keep-alive, chunked transfer, static files

RFC Compliance

This server follows the HTTP specifications defined in:

  • RFC 7230 (HTTP/1.1 Message Syntax and Routing)
  • RFC 9110 (HTTP Semantics)

Project Structure

  • includes/ - Header files
  • srcs/ - Source files
  • www/ - Static files served by the server
  • tests/ - Unit and integration tests
