# webserv

This project is about writing an HTTP server using C++98.
HTTP is one of the most widely used protocols on the internet.
The goal of this project is to create a simple web server that can handle HTTP requests and serve static files.

## Installation

To install the web server, clone the repository and compile the source code using the provided Makefile:

```bash
git clone https://github.com/itaharbo91/webserv.git
cd webserv
make
```
This will create an executable file named `webserv` in the project directory.

## Usage

To run the web server, use the following command:

```bash
./webserv <config_file>
```
Replace `<config_file>` with the path to your configuration file.
